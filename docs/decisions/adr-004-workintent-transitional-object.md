# ADR-004: WorkIntent is a transitional object

## Status
Accepted

## Decision

WorkIntent exists as a lightweight bridge before CaseShyft is fully available.

## Why

Users need to capture:
- follow-up
- assistance
- identity review
- callback needs

without losing signal or inventing side spreadsheets.

## Consequences

- WorkIntent should remain small
- it should support conversion/handoff later
- it must not become a second case-management domain
